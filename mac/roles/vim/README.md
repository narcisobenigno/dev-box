# My sheetcheat

## living without plugins

| command   | descrition                |
| --------- | ------------------------- |
| `e`       | go to the end of the word |
| `gx`      | open a link               |

## Settings (`:set <setting>`)
 
| setting | descrition
| ------ | -----------------
| `hls`  | highlight search
| `ic`   | ignore case
| `is`   | incremental search
| `noic` | disable ignore case
| `in<option> | invert setting

### Turn off setting

| command | descrition |
| ------- | ---------- |
| `:nohlsearch` | remove highlighted matches |
